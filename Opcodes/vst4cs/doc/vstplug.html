vstplug

vst4cs --  Vst plugin hosting in Csound5 on Windows.

Description

vstplug is used for sending and recieving audio from a vst plugin.

Syntax

aout1,aout2 vstplug iflag, ain1, ain2

Initialization

iflag -- the flag which identifies the plugin called previously with a vstinit opcode.

Performance

aout1, aout2 -- the audio output given by the plugin.

ain1, ain2 -- the audio sent to the plug in for processing. For vst instruments input is usually ignored.

Examples


See vstout or vstpsend for examples.

vstinfo

vst4cs --  Vst plugin hosting in Csound5 on Windows.

Description

vstinfo displays the parameters and the programs of a VST plugin.

Syntax

vstinfo iflag

Initialization

iflag -- the flag which identifies the plugin called previously with a vstinit opcode.

The verbose flag in vstinit gives the same information as vstinfo. vstinfo is useful after loading parameter banks, or when plugin changes paramters dynamically.


Performance

Examples


/* orc */
sr = 44100
kr = 4410
ksmps = 10
nchnls = 2

giHandle1 vstinit "c:/vstplugins/cheeze/cheeze machine.dll",1

instr 4
vstbload gihandle1,"c:/vstplugins/cheeze/chengo'scheese.fxb"
vstinfo gihandle1
endin

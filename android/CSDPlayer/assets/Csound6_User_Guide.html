Csound 6 for Android User Guide


Michael Gogins

12 June 2013


This is an extremely basic introduction to using the Csound6 app on Android smartphones, tablets, and other devices. Contemporary smartphones and tablets can be as powerful as personal computers were a few years ago. Despite their small size, they are real computers, and on them, Csound can be used to perform or render real music.


Csound6 is most of Csound, including some of the more frequently used plugin opcodes:


  • The signal flow graph opcodes.

  • The Fluidsynth opcodes for playing SF2 samples.

  • The libstdutil library.

  • The Lua opcodes, which embed LuaJIT and enable arbitrary Lua code to execute in Csound either for signal processing or for score generation or, indeed, for any purpose. Particularly amazing is that LuaJIT has a foreign function interface that is capable of loading any shared library that Android permits, and calling any exported function in that library.


Csound6 has a pre-laid-out user interface consisting of a few menu buttons, five sliders, five click buttons, and a trackpad. In addition, the Android device may provide sensors such as accelerometers, GPS position, and temperature that are assigned Csound control channels.
